JSON5 parser for PHP.This library using [kujirahand/JSON5-PHP](https://github.com/kujirahand/JSON5-PHP) as reference.

## Requirements
- PHP 5.5.11 or later
- Composer
- JSON extension
- mbstring extension

## Features
- Parse JSON5 string
- Parse JSON5 by file
